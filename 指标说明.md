# 评估指标说明

## 基于论文的标准指标

### 1. K-LJP 论文标准（主要指标）

用于评估法律判决预测的准确性：

#### 判决预测准确性
- **Mi-F (Micro-F1)**: 微平均 F1，整体预测准确率
- **Ma-F (Macro-F1)**: 宏平均 F1，每个标签的平均准确率（对罕见标签更敏感）
- **Mi-J (Micro-Jaccard)**: 微平均 Jaccard 系数，预测集合与真实集合的重叠度
- **Ma-J (Macro-Jaccard)**: 宏平均 Jaccard 系数

#### 对齐率（创新点指标）
- **Align (Alignment Ratio)**: 法条和罪名的对齐率
  - 公式：`Align(a,c) = (|a∩C2A(c)| + |c∩A2C(a)|) / (|a| + |c|)`
  - 衡量预测的法条和罪名是否符合法律规定

### 2. PCL 论文（方法论来源，不用其指标）

PCL 提供了 **Test-Time Adaptation** 的方法论：
- 扰动一致性学习
- 测试时模型更新策略

但 PCL 的评估指标（EM/F1）是针对 QA/NER 任务的，不适用于多标签分类。

## 当前实现的指标

### 主要指标（终端显示）
```
Law Article Prediction:
  Mi-F, Ma-F, Mi-J, Ma-J

Charge Prediction:
  Mi-F, Ma-F, Mi-J, Ma-J

Average:
  Avg Mi-F, Avg Ma-F

Alignment Ratio:
  Align
```

### 辅助指标（保存在文件中）
- Precision (精确率)
- Recall (召回率)
- Exact Match (完全匹配率)
- Hamming Accuracy (逐标签准确率)

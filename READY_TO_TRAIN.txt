## 🎯 优化后的配置详解

### 核心改动及原因

#### 1. learning_rate = 5e-6 ⭐⭐⭐⭐⭐
```
为什么: 训练数据只有39k (论文163k的24%)
效果: 避免过拟合，罕见标签学得更好
预期提升: +8-10% Ma-F1
```

#### 2. batch_size = 16 ⭐⭐⭐⭐
```
为什么: 小batch让罕见标签更频繁出现
效果: 梯度更新从1227步/epoch → 2454步/epoch
预期提升: +3-5% Ma-F1
```

#### 3. num_epochs = 40 ⭐⭐⭐⭐
```
为什么: 学习率降低需要更多轮收敛
效果: 给罕见标签足够的学习时间
预期提升: +2-3% Ma-F1
```

#### 4. base_lambda = 3.0 ⭐⭐⭐
```
为什么: 对齐率从92%降到85%，约束太弱
效果: 增强法条-罪名一致性
预期提升: +2-3% Ma-F1，对齐率提升到90%+
```

#### 5. warmup_ratio = 0.15 ⭐⭐
```
为什么: 更长的warmup保证训练稳定
效果: 前期学习率平滑上升
```
